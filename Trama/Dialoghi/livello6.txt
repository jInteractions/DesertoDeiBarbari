OBIETTIVO
L'obiettivo è quello di cambiare il codice contenuto all'interno del meccanismo che si attiva tramite il click del mouse, cioè il contenuto del blocco:
$( planciaComandi ).on( 'click', function () { /* codice da inserire */ } );
Nel secondo file è infatti contenuto l'algoritmo di selezione automatica della torretta, non modificabile. Si deve inserire dunque nel meccanismo attivato dal mouse la chiamata a tale funzione. Questa chiamata necessita di tre prametri:
- torrette, che abbiamo già;
- una x corretta per il bersaglio;
- una y corretta per il bersaglio.
La x e la y rappresentano le coordinate nel piano cartesiano dell'area di sparo delle torrette.
Si deve poi creare una variabile che conterrà il valore restituito da tale chiamata di funzione; poi effettuare un'altra chiamata, questa volta alla funzione azionaComandoSparo(), con i seguenti parametri:
- la stringa 'keyup';
- la torretta selezionata con la chiamata precedente;
- la variabile sistema;
- la x corretta;
- la y corretta.

AIUTI
Un modo per creare il nuovo meccanismo di sparo prevede di inserire nel punto opportuno del codice il corretto valore nelle variabili x ed y:
var x = mirino.x;
var y = mirino.y;
Poi è necessario salvare il valore restituito dalla chiamata alla funzione in una variabile:
var torrettaSelezionata = torrettaVicina( torrette, x, y);
Ed utilizzare tale variabile per effettuare una seconda chiamata di funzione:
azionaComandoSparo( 'keyup', torrettaSelezionata, sistema, x, y );

MANUALE
Chiamata di funzione
Si prenda una funzione di prova, come la seguente:
var calcolaSomma ( a, b ) {
	return a + b;
}
Per effetuare una chiamata a tale funzione, è necessario creare una variabile che contenga il risultato dell'operazione, e inserire tra le parentesi i valori che nella funzione diventeranno a e b:
var risultatoSomma( 4, 5 );
Nella variabile risultatoSomma ora c'è il numero 9.
Area di sparo
L'area di sparo è un piano cartesiano: le coordinate vengono dunque espresse in termini di x e y, che indicano la posizione di un punto all'interno di quest'area.
Il mirino ha dunque una x ed una y, che indicano precisamente dove si trova in quel momento il puntatore collegato al mouse.
