MANUALE
Un altro punto fondamentale nei codici sorgenti del sistema antimissilistico Hob-2000 &egrave rappresentato dalle <b>istruzioni di controllo</b>.
Un'istruzione di controllo si basa su una nuova tipologia di valore che pu&ograve essere inserito in una variabile, chiamato <b>booleano</b>.
Una variabile booleana pu&ograve assumere due valori differenti:
<li> <i>true</i>, se essa rappresenta un concetto vero;</li>
<li> <i>false</i>, se essa rappresenta un concetto falso.</li> <br>
Per rappresentare il vero e il falso &egrave possibile utilizzare sia le parole chiave appropriate (rispettivamente <i>true</i> e <i>false</i>), oppure diversi tipi di operatori:
<li> <tt>a === b</tt>, vero se a &egrave uguale a b;</li>
<li> <tt>a !== b</tt>, vero se a &egrave diverso da b;</li>
<li> <tt>a &#62 b</tt>, vero se a &egrave maggiore di b;</li>
<li> <tt>a &#60 b</tt>, vero se a &egrave minore di b;</li>
<li> <tt>a &#62= b</tt>, vero se a &egrave maggiore o uguale di b;</li>
<li> <tt>a &#60= b</tt>, vero se a &egrave minore o uguale a b;</li>
<li> <tt>!a</tt>, vero se a ha come valore <i>false</i>.</li> <br>
Esempi di variabile contenente un valore vero possono essere:
<pre><code>var b1 = 5 &#62 4;
var b2 = true;
var b3 = !false;
</code></pre>
Esempi di variabile che contengono un valore false possono essere:
<pre><code>var b4 = 3 &#62= 6;
var b5 = false;
var b6 = !true;
</code></pre>
Le istruzioni di controllo hanno bisogno delle variabili booleane. Esistono tre istruzioni di controllo.
La prima &egrave identificata dalla parola chiave <b>if</b>. Eccone un esempio:
<pre><code>var x = 0;
if ( a === true ) {
  var x = 1;
}
</code></pre>
La variabile <tt>x</tt> parte con il valore 0 al proprio interno. A questo punto se <tt>a</tt> &egrave una variabile booleana che contiene un valore vero, allora <tt>x</tt> prender&agrave il valore 1 al proprio interno; altrimenti, cio&egrave se <tt>a</tt> &egrave falsa, <tt>x</tt> non verr&agrave aggiornata perch&eacute la parte di codice all'interno delle parentesi graffe non verr&agrave eseguita.
Come si pu&ograve notare, le istruzioni di controllo sono semafori che, in base al valore che si ha all'interno delle parentesi tonde, decidono l'andamento del codice.
Una seconda tipologia di istruzione di controllo si basa sulle parole chiave <b>if</b> ed <b>else</b>. Eccone un esempio:
<pre><code>var x;
if ( a === false ) {
  x = 1;
} else {
  x = 0;
}
</code></pre>
Questa istruzione descrive lo stesso meccanismo visto nell'esempio precedente: se la variabile <tt>a</tt> &egrave falsa, <tt>x</tt> conterr&agrave il valore 1; altrimenti (<i>else</i>, in inglese), conterr&agrave il valore 0.
L'ultima istruzione di controllo da comprendere &egrave quella chiamata <b>if-else if</b>. Eccone un esempio:
<pre><code>var x = 0;
if ( a === false ) {
  x = 1;
} else if ( b &#62= 4 ) {
  x = 2;
}
</code></pre>
Questa istruzione descrive un meccanismo leggermente differente da quello visto in precedenza. Se la variabile <tt>a</tt> &egrave falsa, allora <tt>x</tt> conterr&agrave il valore 1; altrimenti, cio&egrave se <tt>a</tt> &egrave vera, si controlla se <tt>b</tt> &egrave maggiore o uguale di 4. Se ci&ograve &egrave vero, allora <tt>x</tt> conterr&agrave il valore 2, altrimenti, cio&egrave se <tt>b</tt> &egrave minore di 4, non si fa nulla.
